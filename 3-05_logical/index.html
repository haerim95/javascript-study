&& 연산자 (논리연산자)

왼쪽 조건식 true 오른쪽 조건식 true일때 최종 결과가 true

왼쪽 오른쪽 결과
true true true
true false false
false true false
|| 연산자 (논리연산자)

왼쪽 , 오른쪽 중 하나라도 true라면 최종 결과가 true

단, 왼쪽 오른쪽 조건식이 모두 false여야 최종 결과가 false된다.

왼쪽 오른쪽 결과
true true true
true false true
false true true
false false false
! 조건식

해당 조건식의 결과가 반대로 변경된다.
