push 메서드

()안의 매개변수로 지정한 데이터를 배열의 가장 끝에 추가한다.

배열변수.push('추가할 데이터');

배열 데이터 추가 / 삭제 메서드 목록

메서드명 의미
배열의 변수명.pop() 배열의 마지막 데이터를 삭제한다.
배열의 변수명.push(데이터) 배열의 마지막 부분에 데이터를 추가한다.
배열의 변수명.shift() 배열의 첫 번째 데이터를 삭제한다.
배열의 변수명.unshift(데이터1, 데이터2,…) 배열의 앞 부분에 데이터1, 데이터2,…를 추가한다.
