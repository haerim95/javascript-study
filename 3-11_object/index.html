객체

객체(Object)란?

여러 프로퍼티를 가지고 있는 데이터의 집합

1. 객체작성

중괄호 {}를 사용한다.
var 변수명 = {};

프로퍼티 갯수엔 제한이 없다.

2. 프로퍼티 읽기

[1] 객체명.프로퍼티명

[2] 객체명['프로퍼티명']

3. 프로퍼티 정보 변경

[1] 객체명.프로퍼티명 = 변경할 데이터;

[2] 객체명['프로퍼티명'] = 변경할 데이터;

프로퍼티에 함수 데이터 저장 var obj = {
addTax : function(num){
return num * 1.08;
}
};

프로퍼티의 데이터가 함수 일때 해당 프로퍼티를 '메서드' 라고 부른다.
